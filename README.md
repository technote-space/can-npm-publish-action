## Usage
e.g. `.github/workflows/check-publish.yml`
```yaml
on: pull_request

name: Check npm publish

jobs:
  checkPublish:
    name: Check npm publish
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - uses: technote-space/can-npm-publish-action@v1
```
